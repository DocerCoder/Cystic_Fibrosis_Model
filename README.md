# 🧬 Stratacyst Platinum
**Clinical-Biophysical Integration Engine for CFTR Pathogenicity**

## 🚀 What is Stratacyst Platinum?

The classification of Cystic Fibrosis Transmembrane Conductance Regulator (CFTR) variants remains a significant challenge in precision medicine due to phenotypic discordance—where clinical markers like sweat chloride concentrations do not align with pancreatic or respiratory outcomes. Stratacyst Platinum is a computational engine developed to resolve these gaps. By integrating genomic anchor points with biophysical stability metrics (Mean Stability and S-GFI scores), the engine provides a high-fidelity diagnostic audit.

This project enables clinicians to move beyond standardized labeling by providing a mechanism-based therapeutic recommendation. By identifying the specific molecular defect (e.g., proteostasis failure vs. gating dysfunction), Stratacyst Platinum optimizes the selection of CFTR modulators (Correctors vs. Potentiators), potentially reducing trial-and-error in treatment and improving long-term patient prognosis.

## ✨ Key Features
* **Logic Audit:** Shows the "brain" of the AI so doctors can see why it made a choice.
* **Modulator Matching:** Tells you if a patient needs a "Gate Opener" or a "Folder" medicine.
* **Research Log:** Keeps a history of every test for easy exporting.

## 🛠️ How to Use It
1.  Make sure you have Python installed.
2.  Install the "tools" by typing `pip install -r requirements.txt`.
3.  Run the app by typing `streamlit run your_script_name.py`.
4.  Enter a variant (like **F508del**) and see the magic happen!
