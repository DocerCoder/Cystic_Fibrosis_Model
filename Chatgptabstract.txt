🧬 Project Title

CFTR Modulator Response Predictor: An AI Framework for Personalized Therapy in Cystic Fibrosis

🧠 Abstract

Cystic fibrosis (CF) is caused by mutations in the CFTR gene that disrupt chloride ion transport, leading to multi-organ pathology. While CFTR modulators such as ivacaftor, lumacaftor, tezacaftor, and elexacaftor have revolutionized CF therapy, clinical response varies dramatically between patients—even among those sharing the same variant. This project proposes a novel machine learning framework, the CFTR Modulator Response Predictor (CMRP), designed to forecast the therapeutic efficacy of CFTR modulators for any given variant based on comprehensive multi-domain data.

Using integrated datasets—including CFTR2 variant class, quantitative function scores, patient organoid assay data, sweat chloride levels, pulmonary function (FEV₁), pancreatic status, infection history, BMI, and treatment burden—the model learns to map variant-specific and clinical features to modulator response outcomes. The predictor employs a hybrid deep-learning architecture combining structured numerical data, textual variant annotations, and 3D structural embeddings derived from protein modeling.

The ultimate goal is a precision-medicine tool capable of predicting which drug—or combination of drugs—will restore CFTR function most effectively for each patient. This innovation could dramatically reduce trial-and-error therapy selection, improving outcomes and lowering treatment costs. The research advances CF bioinformatics by integrating structural biology, clinical phenotyping, and AI, forming the foundation for a new era of individualized CF care.

🎯 Purpose and Hypothesis

Purpose: To develop an AI model that predicts a patient’s response to specific CFTR modulators using variant and clinical features.

Hypothesis: CFTR modulator efficacy can be accurately predicted by integrating variant class, molecular function metrics, and patient-level phenotype data into a unified model trained on known modulator response outcomes.

🧩 Research Questions

Can variant class and quantitative function data (e.g., chloride conductance, protein processing efficiency) predict modulator responsiveness?

Do phenotypic measures (sweat chloride, FEV₁, BMI, infection prevalence) improve predictive accuracy beyond genetic data alone?

Can a unified deep-learning architecture outperform traditional regression or ensemble models in predicting modulator outcomes?

🧪 Procedures and Methodology

Data Compilation

Aggregate variant-level data from CFTR2 and literature: variant class, function %, and clinical descriptors.

Integrate patient cohort data (e.g., sweat chloride reduction, FEV₁ improvement, organoid chloride transport) as modulator response targets.

Normalize and encode categorical variables (variant class, organ involvement, infection presence).

Merge with structural embeddings derived from CFTR protein 3D coordinates using AlphaFold-based feature extraction.

Feature Engineering

Quantitative: sweat chloride, BMI, FEV₁, pancreatic status, pseudomonas infection %, function %, protein processing %.

Categorical: variant class (I–VI), organoid phenotype, modulator used.

Structural: per-residue embeddings representing variant location and domain.

Derived: “Therapeutic difficulty index” = inverse of baseline function × infection prevalence.

Model Architecture

Hybrid neural network:

Dense layers for structured data

Transformer or text embedding for variant annotation

Optional 3D CNN/graph module for structural context

Output: Predicted percent improvement in chloride conductance (or sweat chloride reduction).

Training and Validation

Split data into 70% training, 15% validation, 15% testing.

Evaluate using Mean Absolute Error (MAE) and R² for continuous outcomes, accuracy for categorical (responsive vs non-responsive).

Apply SHAP or Integrated Gradients to interpret feature importance.

Visualization

Generate a heatmap correlating predicted drug efficacy across variant classes.

Produce a 3D map of CFTR protein highlighting residues most predictive of modulator response.

🔬 Materials

CFTR2 dataset (variant and clinical data)

NCBI/AlphaFold CFTR structural data

Python (TensorFlow, PyTorch, scikit-learn, Biopython, RDKit, matplotlib, seaborn)

Google Colab or local GPU workstation

⚙️ Expected Results

Accurate prediction of modulator response within ±10% of experimentally observed function for most variants.

Discovery of variant clusters sharing mechanistic response profiles.

Identification of structural “modulator hotspots” on the CFTR protein.

🌍 Significance

This work would be the first CFTR-wide predictive AI capable of recommending optimal modulator therapies for untested or rare variants. It directly addresses a major gap in CF medicine—patients with unknown or variable response—and could guide drug design and clinical decision-making globally.
